# 📱 Flipkart Phones Web Scraping & Data Analysis

## 📌 Project Overview
This project focuses on collecting smartphone data from Flipkart through web scraping and performing data cleaning, preprocessing, and exploratory data analysis (EDA) using Python.

The dataset includes information such as phone prices, ratings, reviews, brand names, RAM, ROM, battery capacity, display size, processor, and camera specifications.

---

## 🛠 Tech Stack
- Python  
- BeautifulSoup  
- Requests  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 📂 Dataset Features
The scraped dataset contains:

- Product Name  
- Price  
- Rating  
- Number of Ratings  
- Number of Reviews  
- Brand  
- Processor  
- RAM & ROM  
- Battery Capacity  
- Display Size  
- Camera Specifications  

---

## 🔍 Project Workflow
### 1. Web Scraping
Collected real-world smartphone data from Flipkart using Python scraping libraries.

### 2. Data Cleaning
Processed raw data by:
- Removing unwanted text and units
- Handling missing values
- Converting columns into proper data types
- Preparing data for analysis

### 3. Exploratory Data Analysis (EDA)
Performed analysis to identify:
- Brand-wise phone listings
- Price trends
- Rating comparisons
- RAM and battery impact on pricing
- Relationships between different phone features

---

## 📊 Visualizations
Created charts and graphs including:
- Bar Charts  
- Histograms  
- Scatter Plots  
- Correlation Heatmaps  

---

## 🎯 Key Outcome
This project demonstrates an end-to-end workflow of **Web Scraping → Data Cleaning → Data Analysis → Visualization** using real-world Flipkart smartphone data and provides meaningful insights into market trends.
